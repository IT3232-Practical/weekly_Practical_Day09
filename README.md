## Getting Started
1. Clone the repository
```bash
git clone https://github.com/IT3232-Practical/weekly_Practical_Day09.git
```

2. Navigate to the project directory
```bash
cd weekly_Practical_Day09
```

3. Build the project
```bash
mvn clean install
```

4. Run the application
```bash
mvn spring-boot:run
```
